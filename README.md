# Projet-TranspoBot-gestion-transport-urbain
Transpobot est une plateforme intelligente de gestion du transport urbain. Elle optimise les itinéraires, suit les véhicules en temps réel, détecte les incidents et aide à la prise de décision grâce à l’IA, améliorant ainsi l’efficacité et l’expérience des usagers.
